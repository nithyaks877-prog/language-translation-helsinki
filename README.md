# 🌐 Language Translator Chatbot

A simple AI-powered Language Translator built using **Streamlit** and **Hugging Face Transformers**. The application translates text from **English** into multiple languages using the **Helsinki-NLP OPUS-MT** translation models.

---

## 🚀 Features

- Translate English text into multiple languages
- Interactive Streamlit web interface
- Powered by Hugging Face Transformers
- Automatically downloads translation models
- Simple and lightweight application

---

## 🌍 Supported Languages

- French 🇫🇷
- German 🇩🇪
- Hindi 🇮🇳
- Spanish 🇪🇸

More languages can easily be added by updating the language dictionary.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/nithyaks877-prog/language-translation-helsinki.git
```

Move into the project folder:

```bash
cd language-translator-chatbot
```

Install the required packages:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

Start the Streamlit app:

```bash
streamlit run app.py
```

The application will open automatically in your browser.

---

## 📸 How It Works

1. Enter English text.
2. Select the target language.
3. Click **Translate**.
4. View the translated text instantly.

---

## 📌 Example

**Input**

```
Hello, how are you?
```

**Target Language**

```
French
```

**Output**

